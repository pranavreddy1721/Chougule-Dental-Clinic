## 📌 Project Overview

This project focuses on designing and developing a **modern and responsive dental clinic website** for **Chougule Multi-Speciality Dental Clinic, Ashta**.

The website provides essential clinic information such as services, doctor details, clinic gallery, location, and contact options through a clean and user-friendly interface.

---

## 🚀 Features

- Responsive and modern UI
- Clinic services information
- Doctor information section
- Clinic gallery
- Google Maps integration for clinic location
- Contact form using EmailJS
- Quick contact options (Call & WhatsApp)

## 🏗️ System Architecture

```
User
 ↓
Internet
 ↓
Clinic Website Frontend (React)
 ↓
APIs (Google Maps / EmailJS)
 ↓
Cloudflare Hosting
```

The user accesses the website through the internet. The frontend displays clinic information while APIs provide location and communication functionality.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/pranavreddy1721/Chougule-Dental-Clinic.git
```

Navigate to project directory

```bash
cd Chougule-Dental-Clinic
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```
